Food Sentiment Analysis (Feb. 2019)

While reading reviews from the business, you might notice that the ratings don't match with the sentiment analysis. That's because an rating number tells one side of the story, but the sentiments of users tell another. Hence, I collaborated with a friend to create a web-app that tells users how people feel about certain food items, based on Yelp Reviews.

• Recognized as the Best Data Analysis Web Application for a business proposal project.
• Parsed Yelp data using Yelp Fusion API and Textblob to provide unique insight on Business reviews.

Is Your Car Safe? (Feb. 2019 - HackUCI)

Do you ever wonder if your car is safe after you parked?

Using Google Maps API and an incident dataset located here, we created a web application that can display incidents near your location and based on the number of incidents, we determine if your parking spot is safe or not!

User can input their location and the application shows them all vehicle-related incidents such as vehicle thefts.

Locations are limited to just San Francisco, CA.

• Collaborated with a team of 4 at HackUCI 2019 to create this functioning application in 3 days.
• Built with Flask (Python Web Framework), HTML/CSS, and JavaScript.

Read more about this project and take a look at how it is implemented by checking out the repository:  github.com/JoshTavasso

Made in collaboration with Justin Leong, David Yip, Joseph Medina

Reddit SMS (Summer 2018)

A Reddit Texting Service 

A user can request information from subreddits to our phone number through their mobile phone and receive a text message containing their desired information.

Purpose: A user can get their latest information from their favorite subreddits in seconds, without needing access to the internet.

Implemented in Python, using Flask (Web Framework), Praw (Reddit API), and Twilio (Texting API).

The demo I have shown represents one example of an interaction. The user uses the following format: <subreddit> <number of posts> <category> and receives a response back with information. Due to limitations with the Twilio free trial, there is a limit to how many characters per text message, so the messages are out of order here and sent in bulk. But, with a Twilio premium account, there is no such limitation.

Read more about this project and take a look at how it is implemented by checking out the repository: github.com/JoshTavasso
